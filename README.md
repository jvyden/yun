# Yun

A general-purpose Discord bot.

## Setting up

1. `git clone https://github.com/jvyden420/yun/`

2. `cd yun`

3. `npm install`

4. `cp .env.example .env`

5. `nano .env`

    **Remember to add your token at this step.**

6. `node yun`
